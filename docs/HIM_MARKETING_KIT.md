# HIM Public Alpha Marketing Kit

## Headline

**Your models are not the whole mind. HIM gives them a brain, memory, tools, and proof.**

HIM is a sovereign multi-model intelligence runtime from ITSNotAI Labs. It routes
work across real model lanes, virtualizes million-token knowledge through Python,
maintains persistent cognitive state, and executes through governed tools with
hash-linked receipts.

Instead of forcing an entire repository or company through one attention window,
HIM keeps the knowledge plane outside the model. Each turn gets a ranked,
source-tagged working set. Every answer can show which model ran, which context
entered, which agents participated, and whether execution was authorized.

## Proof points

- One command launches browser chat and API.
- The million-token logical-context test passes.
- Model identity and parameter accounting are exposed over API.
- Hosted fallback is disabled by default.
- Invalid raw output cannot masquerade as mature generation.
- Read and execution authority are separate.
- Real open weights are included and their current quality is labeled honestly.

## 30-second pitch

“LLMs keep trying to make the model carry the whole system. We separated the
mind from the attention window. HIM combines real model routing, Python-scale
context, persistent brain state, local tools, and cryptographic receipts. Connect
your open weights, teach it a repository, talk in the browser, and see exactly
what produced every answer.”

## Never claim

- HIM-native-v0 is 14B.
- Logical context equals native attention.
- Unrun benchmark accuracy.
- Deterministic fallback is learned generation.
- Proposed work executed without an execution receipt.
