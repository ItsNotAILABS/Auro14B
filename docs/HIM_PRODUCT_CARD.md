# HIM — Sovereign Multi-Model Intelligence Runtime

HIM turns open-weight models, Python context engines, specialist agents, local
browser intelligence, tools, and receipts into one governed intelligence system.

## Working now

- Native and OpenAI-compatible chat APIs.
- Checked-in open-weight tokenizer, weights, loader, and inference path.
- Real model lanes with capability-aware routing and fail-closed fallback.
- Persistent Python/SQLite FTS context virtualization.
- Tested million-token logical storage with bounded injection.
- NOVA model-backed council and fused 44-region cognitive state.
- Governed tools, separate execution authority, and hash-linked receipts.
- Browser chat, context teaching, fleet inspection, and evidence.

## Claim boundary

HIM-native-v0 has 146,576 learned parameters. It is a reference checkpoint, not
Auro-14B and not a broad fluent assistant. Broad chat today requires a promoted
Auro checkpoint behind a compatible endpoint.

The context engine demonstrated a one-million-token logical corpus. Python
selects the active working set; this is not a one-million-token native attention
claim.

Market this release as a public alpha / hackathon-ready sovereign intelligence
runtime, not as a frontier 14B foundation checkpoint.
