# Launch Position

## One sentence

Give every agent a public-web reader you own.

## Launch claim

NEXUS Relay turns public URLs and feeds into clean, cited agent context through one Cloudflare Worker, REST endpoint, and MCP server—with no mandatory monthly scraping-platform subscription.

## Honest comparison

Apify remains a much broader automation platform with Actors, proxies, storage, and managed compute. Monid is a pay-per-use discovery and fulfillment marketplace. Relay's wedge is ownership: deploy a narrow public-web context substrate in your own Cloudflare account, keep the cache and receipts, and add paid providers only when the open path cannot satisfy the request.

## Suggested initial offer

- Open-source core: free.
- Hosted personal endpoint: $9/month plus pass-through usage.
- Team endpoint with API keys, history, watches, and webhooks: $29/month.
- Sovereign/self-hosted commercial support: custom.

These are positioning proposals, not implemented billing plans.
